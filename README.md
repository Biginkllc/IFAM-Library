# IFAM-Library

The IFAM-Library is a thematic catalog of **Ideas For All Mankind (IFAMs)**.

This repo is **not** legal templates. For policies and lease templates, see: **IFAM-Policies**.

## How to use this library
Each IFAM entry is listed by theme (Energy, Education, Water, etc.) and may reference:
- Repomorphing origin logs (innovation entries)
- SIH Harbor narratives (mission framing)
- Optional proof-of-remembrance (IPFS CID + SHA-256)
